# NDVI_project
🌿 NDVI-Based Vegetation Analysis Project

🎯 Aim :
To analyze vegetation in images using a simplified NDVI (Normalized Difference Vegetation Index) technique and estimate green cover using image processing.

🛠️ Tools Used :
Python, NumPy, Matplotlib

📌 Description :
This project processes RGB images to detect vegetation by analyzing pixel values of red and green channels. A simplified NDVI formula is applied to identify green regions in the image. The system then classifies vegetation areas using thresholding and calculates the percentage of vegetation coverage.

⚙️ Methodology :
Load and process RGB image
Extract red and green channels
Compute NDVI-like index using pixel values
Apply threshold to detect vegetation
Visualize original image, NDVI map, and vegetation mask

📊 Output :
NDVI heatmap of image
Vegetation detection map
Percentage of vegetation cover

🚀 Outcome :
Improved understanding of image processing, data analysis, and visualization using Python.
